## 🌟 What Makes SAATHI Different?

### 🧠 AI That Understands YOU
Not generic Western advice. SAATHI's AI engine is driven by deep cultural context — it gets:
- The crushing weight of JEE/NEET preparation
- Parents comparing you to *Sharma ji's* kid
- The fear of *"log kya kahenge"*
- Feeling trapped between dreams and expectations

### 🗣️ AI That Speaks YOUR Language
*"Yaar, bohot stress ho raha hai"* or *"I can't take this anymore"* — SAATHI's AI understands both. Natural **Hinglish**, just like talking to a friend. Powered by **Llama 3.3 70B** for deep linguistic understanding.

### 🚨 AI-Powered Dual-Layer Crisis Detection
When thoughts get dark, SAATHI's AI uses **two layers of protection**:
1. **Keyword bank** — instant AI-driven detection in English, Hindi & Hinglish
2. **Groq AI sentiment analysis** — AI catches indirect hints like *"kya farak padta hai agar main rahoon ya na rahoon"* that no rule-based system can detect

Crisis response includes verified helplines, AI-guided grounding exercises, and auto-call on mobile.

### 🔒 Your Secrets Stay Safe
100% anonymous. No phone numbers. No emails. No judgment. Just pure, private AI-powered support.

## ✨ AI-Driven Unique Features

### 🌿 Mann Ka Haal — AI Mood Tracker
Daily mood check-ins in Hinglish with 5 mood levels, 8 trigger categories, 7-day history graph, and streak tracking. SAATHI's AI reacts empathetically to low moods in real time.

### 🔮 Kal Ka Ankur — AI-Powered Chat with Your Future Self
**The AI-driven feature no other mental health app has.**

Talk to an **AI-powered version of yourself** — 5 years in the future. Your AI future self:
- Knows your exact struggles, dreams, and fears
- Has survived everything you're going through right now
- Speaks in warm Hinglish, like talking to yourself
- Is mysterious but deeply comforting: *"Main itna nahi bata sakta, but trust karo..."*

### 🆘 AI-Enhanced Smart Crisis Panel
- Mobile: 10-second countdown → **auto-dials KIRAN helpline**
- Laptop: Full-screen emergency number display
- 4 verified helplines (KIRAN, Vandrevala, iCall, AASRA)
- Built-in AI-guided 5-4-3-2-1 grounding exercise

## 🛠️ AI-Powered Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React.js + Tailwind CSS |
| Backend | Node.js + Express |

### ⚡ Why Groq + Llama 3.3 70B Powers SAATHI?
- **70B parameters** = superior emotional intelligence and cultural context understanding
- AI catches indirect suicidal hints that rule-based systems and smaller models miss entirely
- AI natively understands Hinglish without translation layers
- **~0.5–1s AI response time** on Groq's LPU hardware — critical for crisis moments where every second counts

## 🚦 Run SAATHI Locally

```bash
# Clone
git clone https://github.com/ankurojha834/saathi.git
cd saathi

# Backend setup
cd saathi-backend
npm install
cp .env.example .env
# Add GROQ_API_KEY to .env

# Run backend
node server.js

# Frontend setup (new terminal)
cd ../saathi-frontend
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) 💚

### Environment Variables

```env
GROQ_API_KEY=your_groq_api_key_here
PORT=5000
NODE_ENV=development
BACKEND_URL=https://your-render-url.onrender.com
```
